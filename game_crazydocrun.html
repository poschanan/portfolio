Crazy Doc. Run

Crazy Doc. Run is 2D platform game developed by Unity. It's my first app! Now, it available in both ios and android.

Story

  Crazy Doc. make a false mistake. Explosion make everyone fall from their star and experimental animals come out. Help Crazy Doc. run to the way back home

  This is project in Game Programming for Mobile Phone Class. Crazy Doc. Run is 2D game in pixel format that has 10 levels now. Character use automatic run system so, player just set direction by click left or right button.

Feature (Version 2.5)

  • Pixel style
  • Automatic run
  • 10 levels
  • 2 characters
