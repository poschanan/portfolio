Your Shoes

  Your shoes is e-commerce website that I create while study Web programming class at university. This website consist of frontend and backend parts. In part of frontend, users can search by keyword to find the shoes that they want. They must register before order products. After that the products were put in bag. Then users can checkout. In part of backend, only admin can login to this section. Admin can add, edit or delete products from stock but still can not edit in order part.
